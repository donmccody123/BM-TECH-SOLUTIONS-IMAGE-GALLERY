## Setup & Installation

1. Clone the repository:
```bash
git clone https://github.com/donmccody123/BM-TECH-SOLUTIONS-IMAGE-GALLERY.git
```

2. Navigate to the project directory:
```bash
cd BM-TECH-SOLUTIONS-IMAGE-GALLERY
```

3. Open `index.html` in your web browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

4. Visit `http://localhost:8000` in your browser

## Features Breakdown

### Slideshow Controls
- **Auto-play**: Images rotate every 5 seconds
- **Manual Navigation**: Click previous/next arrows
- **Dot Indicators**: Click any dot to jump to specific slide
- **Pause on Interaction**: Auto-play pauses when manually navigating

### Lightbox Features
- **Click to Expand**: Click any gallery image to view full-screen
- **Navigation**: Use arrow buttons or keyboard arrows to browse
- **Close**: Click outside image, click X button, or press Escape
- **Image Caption**: Displays descriptive title for each image
